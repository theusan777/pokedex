# Pokedex — Projeto Front-End

Este projeto é uma Pokedex criada para treinar HTML, CSS e JavaScript de uma forma mais visual e prática. A interface permite navegar entre diferentes Pokémon pela lista lateral, e cada seleção exibe um card completo com imagem, tipo, habilidades e estatísticas. O foco foi montar algo organizado, responsivo e com uma UI agradável, sem depender de bibliotecas externas.

## Funcionalidades

• Seleção rápida de Pokémon pela listagem lateral  
• Cards com informações detalhadas  
• Layout responsivo  
• Estilização dividida em múltiplos arquivos CSS  
• Código simples para estudo de manipulação de DOM

## Tecnologias utilizadas

HTML5  
CSS3  
JavaScript  
Google Fonts – Poppins
